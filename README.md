# Dithermark-bun

A headless version of Dithermark using bun.

## Dependencies

* node ~= 16
* bun >= 1.0
* npm

## Getting Started

* `npm install`
* `npm run build`
* `npm start`

## Building with Bun / Running with Node

(Currently doesn't work for some reason - causes segfault in webgl)

* `bun build.js && ./fix-node.sh && node out/index.js`
