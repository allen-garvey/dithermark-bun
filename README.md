# Dithermark-node

A headless version of [Dithermark](https://github.com/allen-garvey/dithermark) using node.

## Dependencies

* node ~= 16
* npm

## Getting Started

* `npm install`
* `npm run build`
* `npm start`

## To update the JSON schemas for options

* `npm run build-types`
