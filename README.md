# Dithermark-bun

A headless version of Dithermark using bun.

## Dependencies

* bun >= 1.0
* npm

## Getting Started

* `npm install`
* `bun src/index.ts`

## Running on Node

* `bun build.js && ./fix-node.sh && node out/index.js`
